01 IoT Connection Guide

1. Hardware Prep

Connect your NodeMCU via USB. Ensure you have the ESP8266 Core installed in your Arduino IDE.

2. Network Auth

Use the Voltiopr credentials provided in your dashboard. The device must connect to the 2.4GHz band to ensure range stability.

3. Endpoint Configuration

// Simple code snippet for the NodeMCU
const char* serverName = "https://api.voltiopr.com/v1/update";

void sendData(float val) {
  HTTPClient http;
  http.begin(serverName); 
  http.addHeader("Content-Type", "application/json");
  int response = http.POST("{\"sensor_val\":" + String(val) + "}");
  http.end();
}

02 Scalable SQL Schema

PostgreSQL / MySQL Optimized

Table: iot_devices

Column Type Description
device_id UUID Primary Key (Unique ID)
last_ping TIMESTAMP Last connection time
status VARCHAR(20) Online, Offline, Fault

Table: telemetry_logs

Optimized for high-frequency inserts (TSDB style)

CREATE TABLE telemetry_logs (
  id BIGSERIAL PRIMARY KEY,
  device_id UUID REFERENCES iot_devices(id),
  reading_type VARCHAR(50),
  reading_value DECIMAL(10,2),
  created_at TIMESTAMPTZ DEFAULT NOW()
);
