Functional Logic

1. Power Control (3 PWM)

Manages three variable output channels. Think of these as "dimmers" that can smoothly adjust power intensity from 0 to 100%.

2. Input Detection (3 Digital)

Monitors three binary states (On/Off). Ideal for mechanical switches, motion sensors, or emergency stop buttons.

3. Analog Sensor (1 Input)

A high-precision ear that listens to varying signals. It translates environmental data (temperature, light levels) into a number the system understands.

Expert Tip

PWM frequency is locked at 1kHz for maximum stability in light-dimming applications.
