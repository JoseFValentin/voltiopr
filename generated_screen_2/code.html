Tabla: Usuarios

Optimizada para Cloudflare D1 (SQLite)

PK
id INTEGER (Auto-Incremental)
NOT NULL UNIQUE
username TEXT (Nombre de usuario)
email TEXT (Correo Electrónico)
password_hash TEXT (Seguridad Encriptada)
google_id TEXT (OAuth Integration)
created_at DATETIME (Registro)

PK ¿Qué es la Llave Primaria?

Imagina el ID como la huella digital única del usuario. En Voltiopr.com, ningún usuario puede tener el mismo número de identificación. Esto permite que el sistema encuentre tu información al instante sin confusiones.

Seguridad de Datos

No guardamos tu contraseña real. Usamos "password_hash", que convierte tu clave en un código secreto imposible de descifrar. Incluso si alguien entrara a la base de datos, vería algo como: $2b$12$R9h/lIPzIZf.O6...

Tecnología de Borde (Edge Computing)

Este esquema está optimizado para Cloudflare D1, garantizando que el inicio de sesión sea ultra rápido desde cualquier parte del mundo.
